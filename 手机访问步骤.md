# 📱 手机访问步骤

## 当前状态
- ✅ 前端服务器：运行在 `https://localhost:5174`
- ✅ 后端服务器：运行在 `http://localhost:3000`
- ✅ 内网穿透服务：已启动

---

## 🚀 快速步骤

### 1️⃣ 查看穿透地址

你应该看到两个命令行窗口：

**窗口1 - 前端穿透（端口 5174）**
```
your url is: https://xxxxx.loca.lt
```

**窗口2 - 后端穿透（端口 3000）**
```
your url is: https://yyyyy.loca.lt
```

### 2️⃣ 配置后端地址

1. **复制后端地址**（窗口2中的地址，如：`https://yyyyy.loca.lt`）

2. **创建或更新 `.env` 文件**（在项目根目录 `F:\06`）

   在 `.env` 文件中添加：
   ```
   VITE_API_URL=https://yyyyy.loca.lt
   ```
   （将 `yyyyy.loca.lt` 替换为你实际的后端地址）

3. **重启前端服务器**
   - 按 `Ctrl+C` 停止当前前端服务器
   - 运行 `npm run dev` 重新启动

### 3️⃣ 在手机浏览器中打开

1. **复制前端地址**（窗口1中的地址，如：`https://xxxxx.loca.lt`）

2. **在手机浏览器中打开**
   - 打开手机浏览器（Chrome、Safari 等）
   - 输入或粘贴前端地址
   - 访问即可

3. **如果提示不安全连接**
   - 点击"高级"或"继续访问"
   - localtunnel 使用自签名证书，这是正常的

## 📲 安装为 PWA（可选）

### iPhone/iPad
1. 用 **Safari** 打开应用
2. 点击底部的**分享按钮**（方框+箭头图标）
3. 选择 **"添加到主屏幕"**
4. 点击 **"添加"**

### Android
1. 用 **Chrome** 打开应用
2. 浏览器会显示 **"安装应用"** 提示
3. 点击 **"安装"**

---

## ⚠️ 注意事项

1. **穿透地址会变化**
   - 每次重启穿透服务，地址可能会变化
   - 如果地址变化，需要重新更新 `.env` 文件并重启前端

2. **保持电脑运行**
   - 内网穿透需要电脑保持运行
   - 关闭电脑后，手机将无法访问

3. **网络要求**
   - 手机需要连接互联网（可以是任何网络，不需要和电脑同一网络）

---

## 🔧 如果看不到穿透窗口

如果看不到穿透地址窗口，可以：

1. **检查任务栏**，可能有最小化的窗口
2. **重新运行穿透命令**：
   ```bash
   # 前端穿透
   lt --port 5174
   
   # 后端穿透
   lt --port 3000
   ```
