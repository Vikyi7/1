```
Forwarding https://xxx.ngrok-free.app -> http://localhost:3000
```

或者打开浏览器访问：`http://localhost:4040` 或 `http://localhost:4042`

### 步骤 2：更新 .env 文件

获取后端地址后（如：`https://xxx.ngrok-free.app`），更新 `.env` 文件：
```
VITE_API_URL=https://xxx.ngrok-free.app
```

### 步骤 3：重启前端服务器

更新 `.env` 后，需要重启前端服务器：
1. 停止当前前端服务器（按 `Ctrl+C`）
2. 运行 `npm run dev` 重新启动
