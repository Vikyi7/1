# 🚀 快速开始 - 第一步：安装 Java JDK

根据检查，你的电脑还没有安装 Java JDK。让我们从第一步开始！

---

## 📥 第一步：下载 Java JDK

### 推荐方式：使用 Adoptium（免费，无需注册）

1. **打开浏览器**，访问：https://adoptium.net/zh-CN/temurin/releases/

2. **选择版本**：
   - **Version**: 选择 **17 (LTS)** 或 **21 (LTS)**
   - **Operating System**: 选择 **Windows**
   - **Architecture**: 选择 **x64**
   - **Package Type**: 选择 **JDK**

3. **点击下载**（文件大小约 180MB）

4. **等待下载完成**

---

## 📦 第二步：安装 Java JDK

1. **找到下载的文件**（通常在 `下载` 文件夹）
   - 文件名类似：`OpenJDK17U-jdk_x64_windows_hotspot_17.0.x_x64.msi`

2. **双击运行**安装程序

3. **安装步骤**：
   - 点击 **"Next"**（下一步）
   - 选择安装路径（默认即可：`C:\Program Files\Eclipse Adoptium\jdk-17.x.x-hotspot`）
   - 点击 **"Next"**
   - 点击 **"Install"**（安装）
   - 等待安装完成（约 1-2 分钟）
   - 点击 **"Finish"**（完成）

4. **记住安装路径**（后面配置环境变量需要用到）

---

## ⚙️ 第三步：配置环境变量

### 3.1 打开环境变量设置

1. 按 `Win + X`，选择 **"系统"**
2. 点击 **"高级系统设置"**（在右侧）
3. 点击 **"环境变量"** 按钮

### 3.2 添加 JAVA_HOME

1. 在 **"系统变量"** 区域（下方），点击 **"新建"**
2. **变量名**：`JAVA_HOME`
3. **变量值**：JDK 安装路径
   - 如果使用默认安装：`C:\Program Files\Eclipse Adoptium\jdk-17.x.x-hotspot`
   - 或者：`C:\Program Files\Java\jdk-17`
   - **注意**：路径中不要包含 `bin` 文件夹
4. 点击 **"确定"**

### 3.3 添加到 PATH

1. 在 **"系统变量"** 区域，找到 **"Path"** 变量
2. 选中 **"Path"**，点击 **"编辑"**
3. 点击 **"新建"**
4. 输入：`%JAVA_HOME%\bin`
5. 点击 **"确定"**

### 3.4 保存并关闭

1. 点击 **"确定"** → **"确定"** → **"确定"**
2. **关闭所有已打开的命令行窗口**

---

## ✅ 第四步：验证安装

1. **重新打开命令行**：
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **输入以下命令**：
   ```cmd
   java -version
   ```

3. **如果看到版本信息**（类似下面），说明安装成功 ✅：
   ```
   openjdk version "17.0.x" 2024-xx-xx
   OpenJDK Runtime Environment Temurin-17.x.x+xx (build 17.0.x+xx)
   OpenJDK 64-Bit Server VM Temurin-17.x.x+xx (build 17.0.x+xx, mixed mode, sharing)
   ```

4. **如果显示错误**（如 `'java' 不是内部或外部命令`）：
   - 检查环境变量配置是否正确
   - 确保路径中没有拼写错误
   - 重新打开命令行窗口
   - 如果还是不行，重启电脑后再试

## ❓ 遇到问题？

### Q: 找不到安装路径怎么办？
**A**: 
- 打开文件资源管理器
- 导航到 `C:\Program Files\Eclipse Adoptium\` 或 `C:\Program Files\Java\`
- 找到 `jdk-17` 或 `jdk-21` 文件夹
- 复制完整路径

### Q: 环境变量设置后还是提示找不到 java？
**A**: 
1. 确保路径正确（不包含 `bin`）
2. 确保 PATH 中添加了 `%JAVA_HOME%\bin`
3. **关闭所有命令行窗口**，重新打开
4. 如果还不行，**重启电脑**

### Q: 可以安装 Java 21 吗？
**A**: 可以！Java 17 或 21 都可以。建议使用 LTS（长期支持）版本。
