# 缘心福 - 极简艺术应用

一款追求极简美学的跨平台艺术应用，采用线条风格设计，参考iOS交互设计规范。

## 特性

- 🎨 **极简设计** - 线条风格，去除冗余
- 📱 **跨平台** - 支持电脑和手机（PWA）
- ✨ **流畅动画** - 参考iOS动画设计，丝滑体验
- 🎯 **优雅交互** - 参考iOS交互设计规范

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- PWA支持

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 设计理念

- 极简线条风格
- iOS风格的动画和交互
- 流畅的用户体验
- 响应式设计
